ENDPOINTS DISPONIBLES (PROTEGIDOS)

Todas las rutas se acceden a través de la base /api.

POST /api/productos

Descripción: Insertar un nuevo producto (requiere idOrigen).

JSON requerido:

{"nombre": "Producto", "idOrigen": 1, "uMedida": "Kilo", "precio": 1000}

Retorno: JSON con el producto creado (código 201).

GET /api/productos

Descripción: Listar todos los productos con el nombre de su mercado.

Parámetros opcionales: ?page=1&per_page=10

Retorno: JSON con lista de productos y metadatos de paginación.

PUT /api/productos/<id>

Descripción: Editar un producto por ID.

Campos opcionales: nombre, idOrigen, uMedida, precio

Retorno: JSON del producto actualizado.

DELETE /api/productos/<id>

Descripción: Eliminar un producto por ID.

Retorno: JSON de confirmación.
